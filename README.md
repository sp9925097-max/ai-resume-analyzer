# AI Resume Analyzer & Job Matcher

## Problem Statement
Students and job seekers struggle to understand whether their skills match job requirements.
Recruiters manually screen resumes, which is time-consuming.

## Solution
An AI-powered web application that analyzes resumes, extracts skills using NLP,
matches them with job descriptions, and suggests missing skills.

## Tech Stack
Backend: Java 17, Spring Boot, Spring Security, JPA, MySQL  
Frontend: React.js  
AI: OpenAI / Gemini API  

## Core Features
- User authentication
- Resume upload (PDF/DOC)
- AI-based skill extraction
- Job matching with percentage score
- Skill gap analysis
